Bike Adventure Program for Awesome Hot Lovely People

Para todos os itens que tiverem escrito apenas o nome de um local assuma que a rota é do homestay até o dado local.

Em Osaka as rotas escolhidas são as que mais tem conduções da grupo JR mas note que tem lugares que não dá pra ir sem pegar transportes de outras empresas, separe dinheiro para esses casos.

Note que em Tóquio o Maps não fornece rotas específicas para bicicletas então essas rotas geradas são para caminhada, por isso:

  • Não se assuste com o tempo de viagem, você vai fazer metade disso
  • Talvez tenham escadarias ou caminhos que são contra-mão no caminho. Fique ligada.
